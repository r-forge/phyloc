caic.table(CAIC)R Documentation

Extract a data frame from a CAIC object.

Description

A CAIC object contains a relatively complex list (obj$contrast.data) containing contrasts and other values. This function extracts this data to a data frame, optionally including information on all nodes in the phylogeny, not just those for which valid contrasts can be calculated. There is also an option to include nodal values for the internal nodes in the data frame.

Usage

caic.table(caicObj, validNodes = TRUE, nodalValues=FALSE)

Arguments

caicObj A CAIC object produced by either caic() or macrocaic().
